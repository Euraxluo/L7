## 简介

- style: 
  - uMin: 123
  - uMax: 123
  - vMin: 123
  - vMax: 123
  - sizeScale: 风场线条粗细缩放比，0-2
  - fadeOpacity: 线条透明度，0-1
  - numParticles: 线条数量
  - rampColors: 线条填充颜色映射，例：
  {
    0.0: '#3288bd',
    0.1: '#66c2a5',
    0.2: '#abdda4',
    0.3: '#e6f598',
    0.4: '#fee08b',
    0.5: '#fdae61',
    0.6: '#f46d43',
    1.0: '#d53e4f'
  }
