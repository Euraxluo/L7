用于在地图上展示 Logo 图片，并且支持超链接点击跳转。

![](https://gw.alipayobjects.com/mdn/rms_816329/afts/img/A*CbdSRLizMLIAAAAAAAAAAAAAARQnAQ)

# API

| 名称 | 说明                                                 | 类型     |
| ---- | ---------------------------------------------------- | -------- |
| img  | Logo 展示的图片 url                                  | `string` |
| href | 点击 Logo 跳转的超链接，不传则纯展示图片，点击不跳转 | `string` |
